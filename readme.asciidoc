== 案例

场景一：
  执行程序能输出“hello world”
  
场景二：输出从 1 到 n 数字的字符串表示。

1. 如果 n 是3的倍数，输出“Fizz”；
2. 如果 n 是5的倍数，输出“Buzz”；
3. 如果 n 同时是3和5的倍数，输出 “FizzBuzz”。

标准解法：

1. 定义变量n
2. 接收从控制台传入的第一个参数，整数
3. 将字符串参数解析成整型
4. 设置循环，变量i，[1,n]
5. 判断3、5整除、输出FixxBuzz
6. 判断3整除，输出Fizz
7. 判断5整除，输出Buzz
8. 其他情况输出i
9. 输出使用控制台输出
